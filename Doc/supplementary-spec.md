# Usability:
- Text and GUI should be readable on the android device.
- Easily accessible GUI and navigation through menus. 
# Reliability- recoverability:
- If the system fails, the user should be able to recover all data from last save point. 
- Restart game with local data.
# Performance:
- application should be able to run smoothly on any android device.
- GUI should respond fast to entice more gameplay.
# Operational concerns:
- Future iterations should log errors.
